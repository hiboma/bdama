# ADR-0001: ビジュアルデザイン方向の決定

## ステータス

承認済み (2026-03-09)

## コンテキスト

幼児向けスマホブラウザパズルゲーム「ビー玉ころころ」のビジュアルデザイン方向を決定する必要があります。10案のデザインバリエーションを CSS モックとして作成し、比較検討しました。

### 検討した10案

1. キャンディポップ - ホットピンク/パープル/イエロー
2. うみのぼうけん - ディープブルー/アクア/サンゴ
3. もりのなかま - フォレストグリーン/モス/アース
4. うちゅうたんけん - ダークスペース/ネビュラ紫/ゴールド
5. サーカスランド - 赤/ゴールド/ロイヤルブルー
6. さくらのくに - 桜ピンク/空色/金
7. レトロゲーム - ネオングリーン/ピクセルブルー/ブラック
8. ゆきのせかい - アイスブルー/フロスト白/オーロラ
9. きょうりゅうランド - ジャングル緑/火山オレンジ
10. おかしのくに - チョコブラウン/ストロベリー/クリーム

## 決定

**05. サーカスランド** を採用します。

### デザイン仕様

- **フォント**: Hachi Maru Pop (Google Fonts)
- **カラーパレット**: ビッグトップ赤 / ゴールド / ロイヤルブルー
- **背景**: クリーム色ストライプ
- **ビー玉**: ゴールドのガラス質感 (CSS radial-gradient)
- **ゴール**: サーカスリング
- **障害物**: ドラム (静的/移動)
- **敵キャラ**: ピエロ (回転ボウタイ)
- **ボス**: リングマスター (シルクハット)
- **演出**: スポットライト効果、回転プラットフォーム

### 難易度構成

| レベル | 要素 |
|--------|------|
| Lv.1 | パス描画のみ |
| Lv.5 | 静的障害物 (ドラム) |
| Lv.10 | 移動障害物 + ピエロ敵 1体 |
| Lv.15 | 狭い通路 + ピエロ 3体 + 回転プラットフォーム |
| Lv.20 | リングマスターボス + 迷路 + 移動障害物 3個 |

## 理由

- サーカスの世界観は幼児にとって馴染みやすく、視覚的に楽しいです
- 赤/ゴールドの配色はコントラストが高く、視認性に優れています
- 障害物 (ドラム)、敵 (ピエロ)、ボス (リングマスター) のキャラクター性が明確で、幼児が直感的に理解できます
